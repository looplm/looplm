# Security Policy

## Supported Versions

We currently support:

- `main` branch
- The latest tagged release

## Reporting a Vulnerability

Please report security issues by emailing `timtreskatis@gmail.com` with details and reproduction steps.

If you prefer, you can also use GitHub Security Advisories for private disclosure.

## Response SLA

- Acknowledgement within 72 hours
- Status update within 7 days
- Fix or mitigation within 30 days when feasible

## PGP

No PGP key is published yet.
